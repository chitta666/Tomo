# Tomo’s Personal Lab

AI-powered automation. Prototype fast. Validate faster.

This repo is my personal workspace for building and documenting practical automations and small prototypes.

## What’s inside
- /gbp-automation/        Google Business Profile: review replies / post automation
- /workflows/             Zapier・Power Automateのフロー設計、プロンプト、実験ログ
- /prototypes/            Small tools (spec → break down → validate)
- /templates/             Reusable templates (specs, checklists, prompts)
- /notes/                 Decisions & learnings

## Current focus
- GBP automation:
  - review auto-reply (⭐4–5, delay, tone control)
  - context-aware posting (rain/holiday/busy season)
- Practice app building with Git:
  - build a simple “evaluation app” to learn specs, decomposition, validation, and Git workflow

## Workflow (my loop)
1) Write spec + acceptance criteria
2) Break down tasks
3) Build minimum prototype
4) Validate with checklist
5) Improve or discard fast

## Rules
- No secrets (API keys / tokens) or personal/customer data.
- Keep commits small and descriptive.
- Each prototype should include:
  - spec.md
  - checklist.md
  - notes.md (what worked / what didn’t)

## Tools
Zapier / Power Automate / Excel / ChatGPT
